THE DATE-A-BASE

This is a website that utilizes 3 APIs to automatically and randomly generate date night ideas! Using a simple drop-down UI, a user can choose their preferences for movies, meals, and drinks and quickly get a date night generated! The best part is, with over 800,000 options, you'll never get the same date more than once!

- TECHNOLOGIES USED -

  • HTML / CSS / JavaScript / APIs

TOWN CRIER

Town crier was made with the intention of returning social media to a more localized scene. Users are able to make posts and comments in a forum that is restricted to their city. Users can learn about events and happenings around their area.

THIS WEBSITE!

This website was made completely by me using no templates or codepens! I have always loved the look of old CRT monitors and wanted my website to feel like a step back into the past. The styling and color theme was based off the display of the old Apple Monitor III from 1980! I hope you enjoy!
