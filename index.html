THE DATE-A-BASE

This is a website that utilizes 3 APIs to automatically and randomly generate date night ideas! Using a simple drop-down UI, a user can choose their preferences for movies, meals, and drinks and quickly get a date night generated! The best part is, with literally millions of combinations, you'll never get the same date more than once!

- TECHNOLOGIES USED -

  • HTML / CSS / JavaScript / APIs
